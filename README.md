# 📊 Business Performance Analytics using SQL

## 📌 Project Overview

This project demonstrates how SQL can be used to transform raw business data into meaningful insights that support data-driven decision-making.

The project focuses on exploring customer, product and sales data to identify trends, evaluate performance, and uncover opportunities that businesses can use to improve operational and strategic decisions.

Through a series of analytical SQL queries, I performed data exploration, segmentation, trend analysis and business reporting to answer key business questions.

---

## 🎯 Project Objectives

- Explore and understand the structure and quality of business data
- Analyse customer, product and sales performance
- Identify trends and patterns over time
- Segment customers and products based on performance
- Generate business-focused reports
- Support data-driven decision-making
