Title

Business Performance Analytics using SQL

Project Overview

This project demonstrates SQL-based exploratory and advanced analytics techniques to analyse customer, product and sales performance data. The objective was to transform raw data into actionable business insights that support data-driven decision-making.

Skills demonstrated
SQL querying
Data exploration (EDA)
Trend analysis
Customer segmentation
Product performance analysis
Ranking analysis
Cumulative analysis
Business reporting
Data-driven decision making
Business questions answered
Which products perform the best?
Which customer segments generate the highest revenue?
How has performance changed over time?
Which areas require improvement?
What trends can support future business decisions?
